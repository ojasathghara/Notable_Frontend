## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

-   A good computer with Node JS 18.0 or higher and npm 8.8 or higher installed.
-   A web browser.
-   A Terminal app to execute commands.
-   A code editor if you want to edit the source code.

### Installation

1. Clone and start the server from: [Notable Backend](https://github.com/ojaswi825/Notable_Backend)
2. Clone this repo
    ```sh
    git clone https://github.com/ojaswi825/Notable_Frontend.git
    ```
3. Install NPM packages

    ```sh
    cd Notable_Frontend
    npm install
    ```

4. Run on localhost
    ```sh
    npm run start
    ```
